## 💬 Task 2: Enterprise AI FAQ Chatbot Dashboard

A highly responsive, executive-style conversational dashboard featuring an optimized, client-side Natural Language Processing (NLP) intelligence matrix. Built to fulfill **Task 2 (Chatbot for FAQs)** of the CodeAlpha AI Internship track.

### 🧠 Under the Hood: How the NLP Engine Works
Instead of relying on heavy cloud servers or paid API backends, this chatbot runs a custom **Zero-Dependency JavaScript NLP Architecture** directly inside the client's browser engine:
1. **Tokenization & Text Cleansing:** The script dynamically captures user strings, forces downcasing, strips punctuation arrays, and splits inputs into distinct token indexes.
2. **Grammar Stop-Word Filtering:** Common grammatical connecting tokens (e.g., *is, the, an, with*) are isolated to prevent skewing the matching telemetry.
3. **Algorithmic Jaccard Set Similarity:** Compares input token intersections against the target question sets to calculate raw structural alignment.
4. **Intent Keyword Weight Matrix:** Pairs token matches with a heavy-weighting bonus tracking array for unique industry terms (e.g., *submission, deadline, YOLO, LSTM*), establishing a highly accurate mathematical **Match Confidence Score**.

---

### ✨ Dashboard Features
* **Real-Time Performance Dashboard:** An active analytical sidebar tracking calculated runtime response latency (ms) and accumulated parsed token metrics.
* **Semantic Target Matching:** Successfully resolves intent across an expanded 10-vector knowledge base mapping documentation, deadlines, and technical task details.
* **Text-to-Speech (TTS) Engine:** Integrated Web Speech API framework reading model outputs aloud via native browser accent vectors for accessibility.
* **One-Click Replicators & Action Layers:** Embedded text clipboard macros and custom typing loaders mimicking live cloud server computations.
* **Automated Fallback Gateway:** A secure confidence boundary check (25% match minimum gate) that automatically serves fallback redirect parameters for out-of-scope prompts.

### 🛠️ Core Architecture Stack
* **UI Interface:** HTML5, Tailwind CSS CDN Framework, FontAwesome Core v6.4.0
* **Algorithmic NLP Layer:** Pure Asynchronous JavaScript Logic
* **Audio Telemetry:** Web Speech API (`SpeechSynthesisUtterance`)
